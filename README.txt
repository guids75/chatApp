To build and launch the app, you can use the command :
mvn spring-boot:run

To use the app, you have to interact with the console :
- The app will ask you your username : just type it in the console and press Enter
The username must be with 20 characters maximum.
The username must be not already taken.

Then, you have 2 actions possible :
- Type the word "send" to send a message :
    The app will then ask you what message you want to send : just type it in the console and press Enter
    This message will be sent to the others users
- Type the word "disconnect" to disconnect from the chat : your session will be closed, and your username forgotten
